非常に重い原子核の安定な構造に関する研究

概要
中性子星表面のような天体には、地上には存在しない非常に重い原子核が存在します。 その形状について、球対称性を保ちつつも中央付近に密度の薄い領域が生じるバブル型構造をとる形状や 球対称性が破れトーラス型の形状を持つと予言している研究があります。 特定の条件下を指定し、その条件下でこれらの構造がどのようなふるまいをするのかを調べます。
質量公式とは
液滴模型に関して、原子核の結合エネルギーを表す簡単な公式として考案されたものです。その結合エネルギーは体積項・表面項・非対称項・クーロン項の4つの項で表されます。
この質量公式では暗黙に一様球を仮定しています。この原子核の半径はr0A1/3であらわされます。また、r0 ~ 1.24で扱われることが多いです。
体積項 : -b V A 表面項 : b S A 2 3 非対称項 : b A (N-Z) 2 A クーロン項 : b C Z 2 A 1 3
ここでは、一様球の仮定を含んで考えていたので、ここからは一様球の仮定を取り除いた標識を考えます。その結果、次のように表記します。
体積項 : - β V A 表面項 : β S Σ 非対称項 : β A ( N-Z A ) 2
クーロン項 : e 2 2 ∫ ρ p ( r i ) ρ p ( r j ) | r i - r j |
この式からわかるように、質量数A、陽子数Zがわかっている状態であればクーロン項の積分が計算できれば、解析的にエネルギー値を算出することができます。 しかし、積分計算としてよく知られる台形則などは r i と r j が同一の位置にある際にクーロン項が発散してしまうことがわかるので、この発散の処理を考える必要があります。 そこで、私はモンテカルロ積分という解析手法を用いて積分することで発散を避けることができると考えました。以下ではモンテカルロ積分の概要に関して説明します。
モンテカルロ積分とは
モンテカルロ積分とは、モンテカルロ法に基づいた積分を行います。モンテカルロ積分では区分求積法のような考え方を用います。
区分求積法は以下の公式が一般的に知られています。
lim n → ∞ ∑ k = 0 n-1 f( k n ) 1 n = ∫ 1 0 f( x )d x
ここの区分求積法からモンテカルロ積分へ応用します。任意の範囲に一様乱数を生成し、積分領域内に含まれる乱数が発生したときは count し、その時のf(x)も加算していく。試行が終了した際に、 α f(x) count を計算することで積分結果を得ることができます。ここでのαは積分、区間・面積・体積に対応します。
以上の手法を用いて本研究に取り組みました。
